| Command                   | Hotkey1  | Hotkey2     |
| :------------------------ | :------- | :---------- |
| Next tab                  | Alt + l  | tab         |
| Prev tab                  | Alt + h  | shift + tab |
| Scroll up                 | k        |             |
| Scroll down               | j        |             |
| Scroll top                | g        |             |
| Scroll bottom             | G        |             |
| Select Item               | Space    |             |
| Toggle Focus Pane         | Tab      |             |
| Toggle Sidebar Visibility | Ctrl + b |             |
| Toggle Bookmark           | Ctrl + d |             |
